# Undertone

A marketing website for Undertone — push-to-talk dictation that turns speech into polished text at your cursor.

## Stack

- **Vite** — build tool
- **TypeScript** — type-checked source
- **GSAP** — scroll animations

## Pages

| File | URL |
|---|---|
| `index.html` | `/` — Landing page |
| `pricing.html` | `/pricing` — Pricing plans |
| `privacy.html` | `/privacy` — Privacy policy |
| `terms.html` | `/terms` — Terms of service |

## Dev

```bash
npm install
npm run dev       # local dev server
npm run build     # production build → dist/
npm run preview   # preview the production build
```

## Deploy

Deployed automatically to [Vercel](https://vercel.com) on every push to `main`.
